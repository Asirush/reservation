# WebApi

WebAPI-example is a backend application, which has several methods:
- Add Reservation
- Delete Reservation
- Edit Reservation

## How to run project

Run reservation/WebAPI-example/WebAPI-example.sln

## Usage

```
Application will open two pages:
- swagger for api
- web page for reservations
```
